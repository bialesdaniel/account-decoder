# account-encoder


### Prerequisites
Install python3
```bash
brew install python
```
Install virtualenv
```bash
pip install virtualenv
```
Create a virtual environment.
```bash
#in this directory
virtualenv venv
source venv/bin/activate
```
### Install
```
pip install -r requirements.txt

```


### Decode
```
python3 cli.py decode -a "<encoded number>"
```
